Service programs

Multi-day and scheduled service programs. All gated by the master switch Control master — you must enable both the master and the per-program switch for anything to actually run. The status cards below show the current state at a glance.

Programs write directly to the heat pump via the configured control write-helpers (see Configuration → Control write targets). If a program does nothing when triggered, double-check those helpers point at existing entities.
